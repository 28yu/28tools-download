PDF → Excel 構造図抽出

🔒 プライバシー保護 — PDF はあなたのブラウザ内のみで処理されます。サーバへ送信されません。 難読化PDFのみ初回 Tesseract.js (≈16MB) を遅延ロードします。
⚠️ 自動読み取りの精度について — この機能は完璧ではありません。 抽出結果は必ず元 PDF と照合し、誤りはプレビュー画面 (黄色セル) で修正してください。
  • PDF にテキスト情報が無いもの (スキャン画像 / フォント難読化) は画像認識(OCR)で読み取るため、誤認識が発生します。
  • 図面ごとにレイアウトが微妙に異なるため、想定外の位置にある情報は拾えない場合があります。
  • 注記文 (「特記なき限り…」など) は、文脈によって認識精度が変わります。
  • 文字が小さい / かすれている / 線と重なっている場合は OCR の精度が落ちます。
