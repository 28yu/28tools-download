Revitでシートを一括作成する方法

無料アドインで図枠を指定して複数シートをまとめて作成

Revit でシートを作るとき、「シートを新規作成 → タイトルブロックを選ぶ → 番号と名前を入力」を1枚ずつ繰り返していませんか？枚数が多いと、この作業だけでかなりの時間がかかります。この記事では、無料アドインで複数のシートを一括作成する方法を解説します。

1枚ずつ作るのが大変な理由

  • 手数が多い：シート1枚ごとに「新規作成→図枠選択→番号入力→名前入力」の操作が必要。
  • 番号・名前の付け間違い：手入力を繰り返すと、連番のミスや表記ゆれが起きやすい。
  • 枚数が多いほど時間を圧迫：数十枚規模になると、シート作成だけで作業時間を大きく消費する。

無料アドインで一括作成する手順

28 Tools の Revit アドインには、図枠を指定してシートをまとめて作成する機能があります。基本の流れは次の4ステップです。

  1. 無料アドインを導入 — 28 Tools のアドインをダウンロードしてインストール
  2. 図枠（タイトルブロック）を選ぶ — 作成するシートに使うタイトルブロックを選択
  3. シート番号・シート名を指定 — 作成したいシートを一覧で指定
  4. 一括作成を実行 — 指定した分のシートがまとめて作成される

⬇️ 無料アドインをダウンロード

アドインの導入方法

この機能は 28 Tools の Revit アドインに含まれています。アドインページから、お使いの Revit のバージョン（2021〜2026）に合ったファイルをダウンロードしてインストールしてください。インストール後、リボンに「28 Tools」タブが追加されます。

もっと便利に使うヒント

  • 命名ルールを先に決める — シート番号の付け方（例：A-01, A-02…）を決めてから一括作成すると整理しやすい。
  • タイトルブロックは事前に読み込む — 使いたい図枠ファミリをプロジェクトに読み込んでおくと選択がスムーズ。
  • 作成後の割付とセット運用 — 一括作成でシートの器を用意し、ビューの割付は続けて行うと効率的。

うまく作成できないときは

症状 対処
図枠が選択肢に出てこないタイトルブロックファミリがプロジェクトに読み込まれているか確認してください。
シート番号が重複してエラーになる既存シートと同じ番号は使えません。重複しない番号を指定してください。
作成したシートが見当たらないプロジェクトブラウザの「シート」ツリーを確認してください。番号順に並んでいます。

よくある質問（FAQ）

アドインは無料ですか？

はい、無料でダウンロードして使えます。Revit 2021〜2026に対応しています。

何枚まで一括作成できますか？

指定した数のシートをまとめて作成できます。多数のシートも一度の操作で作成可能です。

作成後にシート番号や名前は変更できますか？

はい。作成後もRevitの通常操作でシート番号・シート名を編集できます。

任意のタイトルブロックを使えますか？

はい。プロジェクトに読み込まれているタイトルブロックファミリを選んで作成できます。
