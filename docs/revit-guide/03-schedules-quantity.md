# 集計表（明細表）を使って数量を自動集計する

Revit の集計表（スケジュール）機能を使うと、モデル内の要素情報を自動的に一覧化し、数量・面積・体積などを瞬時に集計できます。モデルを更新すると集計表も自動で更新されるため、手作業による数量拾いのミスや手間を大幅に削減できます。この記事では集計表の作成から Excel エクスポート、シートへの配置までを解説します。

---

## この記事でわかること

- 集計表（明細書/数量）の新規作成手順
- フィールド・フィルター・並べ替え・グループ化の設定方法
- 計算値（算出フィールド）の追加方法
- マテリアル数量集計表の作成方法
- 集計表を Excel にエクスポートする方法
- シート（図面）への集計表の配置方法

---

## 手順

### ステップ1: 集計表を新規作成する

1. 「**ビュー**」タブ → 「**作成**」パネル → 「**明細表**」ドロップダウン → 「**明細書/数量**」をクリック。
2. 「新しい明細書」ダイアログが開く。左側の「カテゴリ」リストから集計対象を選択する。

   **よく使うカテゴリの例**

   | カテゴリ | 集計できる主な情報 |
   |---|---|
   | ドア | 枚数・幅・高さ・タイプ・ファイア等級 |
   | 壁 | 面積・体積・タイプ・仕上げ材 |
   | 部屋 | 面積・周長・体積・床仕上げ・天井高 |
   | 床 | 面積・体積・タイプ |
   | 構造フレーミング | 長さ・断面タイプ・材質 |
   | 機械設備 | 個数・タイプ・メーカー情報 |

3. 「名前」欄に分かりやすい名前を入力（例: `ドア集計表`、`壁面積集計`）。
4. 「フェーズ」を必要に応じて設定（新築のみ集計する場合は「新規構築」など）。
5. 「OK」をクリックすると「明細書のプロパティ」ダイアログが開く。

---

### ステップ2: フィールド（列）を設定する

「明細書のプロパティ」の「**フィールド**」タブで、集計表に表示する列を選択します。

1. 「使用可能なフィールド」リストから追加したいパラメータを選択し、「**追加 →**」をクリック。  
   または、ダブルクリックで追加可能。
2. 「明細書フィールド」に追加された列の順序は、上下ボタンで変更できる。
3. 集計表に表示したくないが計算に使いたいフィールドは、後述の「計算値」を活用する。

**集計表でよく使うフィールドの組み合わせ（ドアの例）**

| フィールド名 | 説明 |
|---|---|
| タイプマーク | ドア種別の記号（A-01 など） |
| 幅 | 開口幅（mm） |
| 高さ | 開口高さ（mm） |
| 合計 | 同一タイプの枚数（カウント） |
| レベル | 設置階 |
| コメント | 備考欄 |

> **ヒント**: 「使用可能なフィールド」のドロップダウンを変更すると、カテゴリのシステムパラメータだけでなく、追加した共有パラメータやプロジェクトパラメータも表示されます。共有パラメータの設定方法は [共有パラメータとプロジェクトパラメータの使い分け](./02-shared-parameters.md) を参照してください。

---

### ステップ3: フィルターを設定する

「**フィルター**」タブで集計対象を絞り込みます。条件は最大 8 つまで AND 条件で設定できます。

1. 「フィルター対象」ドロップダウンでフィルターに使うフィールドを選択。
2. 条件（「次と等しい」「次を含む」「次より大きい」など）と値を入力。

**フィルターの設定例**

| ユースケース | フィルター対象 | 条件 | 値 |
|---|---|---|---|
| 特定のレベルのみ集計 | レベル | 次と等しい | 1F |
| 一定幅以上のドアのみ | 幅 | 次より大きいか等しい | 900 |
| 特定のタイプを除外 | タイプ名 | 次を含まない | テスト |

---

### ステップ4: 並べ替えとグループ化を設定する

「**並べ替え/グループ化**」タブで、集計表の行の順序とグループ化を設定します。

1. 「並べ替えの基準」を最大 4 段階まで設定できる。
2. 「**ヘッダー**」「**フッター**」にチェックを入れると、各グループの先頭・末尾に小計行を追加できる。
3. 「**すべてのインスタンスをアイテム化する**」のチェックを外すと、同じタイプをまとめてカウント表示（タイプ集計形式）になる。

**並べ替え設定の例（ドア集計）**

| 優先度 | 並べ替え基準 | 方向 | ヘッダー | フッター（小計） |
|---|---|---|---|---|
| 1 | レベル | 昇順 | ✓ | ✓ |
| 2 | タイプマーク | 昇順 | — | — |

---

### ステップ5: 書式設定をカスタマイズする

「**書式**」タブでは各フィールドの表示形式を調整します。

1. 左側のフィールド一覧から対象を選択。
2. 「**フィールドの書式**」ボタンで数値の単位・小数点以下の桁数・接頭辞・接尾辞を設定。
3. 「**合計を計算する**」にチェックを入れると、そのフィールドの合計値が集計表下部に表示される。  
   数量・面積・長さなどの数値フィールドに有効。
4. 「**非表示フィールド**」にチェックを入れると、集計表には表示されないが並べ替えやフィルターには使えるフィールドになる。

---

### ステップ6: 計算値（算出フィールド）を追加する

既存のパラメータから計算式で新しい列を作成できます。

1. 「**フィールド**」タブで「**計算値**」ボタンをクリック。
2. 名前・規律・タイプを設定する（例: 名前=`開口面積`、タイプ=`面積`）。
3. 「**式**」欄に計算式を入力する。

**計算値の式の例**

| 計算値名 | 式の例 | 説明 |
|---|---|---|
| 開口面積 | `幅 * 高さ` | ドアの開口面積（幅×高さ） |
| 単価合計 | `単価 * 合計` | 数量 × 単価（単価はプロジェクトパラメータで追加） |
| 壁面積（純）| `面積 - 開口部面積` | 仕上げ計算用の正味壁面積 |

> ※Revit 2021 以降、計算値の式でより複雑な条件分岐（`IF` 関数）が利用できるようになりました。例: `IF(幅 > 1800, "大型", "標準")`

---

### ステップ7: マテリアル数量集計表を作成する

壁・床・屋根などの複合レイヤーに使われているマテリアルの体積・面積を集計するには、専用の「マテリアル数量集計」を使います。

1. 「**ビュー**」タブ → 「**作成**」パネル → 「**明細表**」→ 「**マテリアル数量集計**」をクリック。
2. カテゴリを選択（例: 壁）→「OK」。
3. 「フィールド」タブで以下のようなフィールドを追加する：

   | フィールド | 内容 |
   |---|---|
   | タイプ | 壁タイプ名 |
   | マテリアル: 名前 | 各レイヤーのマテリアル名 |
   | マテリアル: 体積 | そのマテリアルの体積合計 |
   | マテリアル: 面積 | そのマテリアルの面積合計 |

4. 並べ替え・フィルターを設定して「OK」。

> **活用例**: コンクリートやモルタルの体積集計、断熱材の面積集計など、積算業務での数量拾いに活用できます。

---

### ステップ8: 集計表を Excel にエクスポートする

集計結果を Excel や他のアプリケーションで加工する場合は、テキスト区切りファイルとしてエクスポートします。

1. 集計表ビューを開いた状態で、「**ファイル**」タブ → 「**エクスポート**」→「**レポート**」→「**明細書**」をクリック。
2. 保存場所とファイル名を指定（拡張子は `.txt`）。
3. 「明細書をエクスポート」ダイアログで以下を設定：

   | オプション | 推奨設定 |
   |---|---|
   | タイトルの行数を含む | チェックあり |
   | 列ヘッダーを含む | チェックあり |
   | テキスト修飾子 | `"（ダブルクォーテーション）` |
   | 区切り文字 | タブ区切り（Excel で開きやすい） |

4. 「OK」でエクスポート。
5. Excel で開くには、ファイルを右クリック → 「プログラムから開く」→「Microsoft Excel」を選択するか、Excel の「データ」タブ → 「テキストから」でインポートする。

> **28 Tools のアドインを使う場合**: 28 Tools の Excel 連携機能を使うと、集計表データの取り込みや書き出しをより効率的に行えます。詳細は [Excel との連携（データのインポート・エクスポート）](./04-excel-integration.md) を参照してください。

---

### ステップ9: 集計表をシート（図面）に配置する

集計表は通常のビューと同様に、シートに配置して図面として印刷・出力できます。

1. プロジェクトブラウザで対象の集計表ビューを確認。
2. 「**ビュー**」タブ → 「**シート構成**」パネル → 「**シート**」で対象のシートを開く（または新規作成）。
3. プロジェクトブラウザの集計表ビューをシートのキャンバスに**ドラッグ＆ドロップ**。
4. 集計表の配置位置をクリックして確定。
5. 集計表の外枠（ビュータイトル枠）をクリックし、プロパティで「ビュータイトルの表示」などを調整。

**集計表の分割表示（長い集計表の場合）**

集計表が 1 シートに収まらない場合は、シートに配置後に「**分割**」ハンドル（集計表右下の青い矢印）をドラッグして複数列に分割できます。

---

## Tips

- 集計表のセルは直接編集できる。プロパティパレットを経由せずに、集計表ビュー上でインスタンスパラメータの値を直接入力・修正できるため、大量のデータ入力に便利。
- 「**ビュー**」タブ → 「**作成**」→「**明細表**」→「**ノートブロック**」を使うと、モデルに配置した注釈記号（タグ）を一覧化した集計表を作れる。
- 集計表ビューのヘッダー行を右クリック → 「**列のサイズ変更**」で各列幅を数値指定できる。シート配置後のレイアウト調整に使う。
- 集計表の「書式」タブで**背景色**を設定すると、特定の列をハイライトできる。※Revit 2022 以降で利用可能。
- プロジェクトブラウザで集計表を右クリック → 「**複製**」すると、設定を引き継いだ新しい集計表を作れる。フィルター条件だけ変えた複数バージョンの集計表を作るときに便利。
- 集計表の行をクリックすると、モデル内の対応する要素がハイライトされる（逆に要素を選択すると集計表内の該当行がハイライトされる）。モデルと集計表を行き来して確認作業をするときに便利。

---

## 注意事項

- 集計表の「**合計を計算する**」はフィールドの単位と規律が一致している場合のみ正しく動作する。単位の異なるフィールドを誤って合計しないよう注意。
- マテリアル数量集計はモデルの精度に依存する。コア境界の設定が不正確な場合、体積・面積の計算結果が実際の設計と異なる場合がある。
- 「すべてのインスタンスをアイテム化する」をオフにしてタイプ集計にした場合、特定のインスタンスを選択して集計表にハイライトする機能が制限される。
- Excel エクスポートは Revit 集計表の書式（色・フォント・合計行）は引き継がれない。Excel 側で別途書式設定が必要。
- フェーズフィルターを適切に設定しないと、既存躯体や解体予定の要素が集計に混入することがある。フェーズの考え方については [フェーズ管理で既存・改修・新築を管理する](./06-phase-management.md) を参照。
- ワークシェアリング環境では、他のメンバーがローカルコピーに未同期の変更を持っている場合、集計表の数値が最新ではない可能性がある。正確な集計が必要な場合は全員に同期を依頼してからセントラルモデルで確認すること。
