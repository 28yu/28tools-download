# 壁・床・屋根の作り方

Revitでのモデリングは、CADで線を引くのとは根本的に考え方が異なります。壁・床・屋根はそれぞれ「建築要素」として作成し、素材・厚さ・高さといった情報を持ちます。この記事では、建物の基本要素である壁・床・屋根の作成手順を解説します。

---

## この記事でわかること

- 壁のタイプ選択・高さ設定・配置方法
- 床のスケッチモードを使った作成手順
- 屋根の「フットプリント形式」と「押し出し形式」の違いと使い分け
- Join Geometry（要素の結合）で壁の交差部をきれいに処理する方法

---

## 手順

### ステップ1: 壁を作成する

#### 1-1. 壁コマンドを起動する

1. プロジェクトブラウザで平面図ビューを開く（例：1階平面図）
2. 「**建築**」タブをクリック
3. 「**ビルド**」パネルの「**壁**」ボタンをクリック
   - 「壁: 建築」と「壁: 構造」が選べる。仕上げ壁は「壁: 建築」を使用する

#### 1-2. 壁のタイプを選択する

1. 「修正 | 壁を配置」タブが自動表示される
2. プロパティパネル上部の**タイプセレクタ**をクリック
3. ドロップダウンから使用したい壁タイプを選択する
   - 例：「基本壁: 外壁 - レンガ」「基本壁: 内壁 - 150mm」など
4. タイプを変更したい場合は「タイプ編集」ボタンをクリックして厚さや材質を設定する

#### 1-3. 壁の高さと配置レベルを設定する

プロパティパネルで以下を設定する:

| プロパティ | 説明 | 例 |
|-----------|------|-----|
| 位置線 | 壁のどの面を基準に描くか | 壁の中心線・外面・内面など |
| 拘束（ベース） | 壁の下端レベル | レベル1（1階） |
| ベースオフセット | ベースレベルからのオフセット距離 | 0 |
| 非接続高さ | 壁の高さ（レベル未指定の場合） | 3000（mm） |
| 上端制約 | 壁の上端レベル（レベルに接続する場合） | レベル2（2階） |

#### 1-4. 壁を描く

1. オプションバー（リボン下の帯）で「直線」「矩形」「円弧」などの描画モードを選択
2. ビューウィンドウで壁の起点をクリック
3. 終点をクリックして壁を配置
4. Escキーを2回押してコマンドを終了する

**連続した壁を描くには**: 起点→第1角→第2角と連続してクリックしていく。矩形を一度に描くには、オプションバーの「矩形」を選択して対角2点をクリックする。

---

### ステップ2: 床を作成する

床はスケッチモードで輪郭（境界線）を描いて作成します。

#### 2-1. 床コマンドを起動する

1. 平面図ビューを開く
2. 「**建築**」タブ → 「**ビルド**」パネル → 「**床**」→「**床: 建築**」をクリック
3. スケッチモードに切り替わり、リボンが「修正 | 床の境界を作成」に変わる

#### 2-2. 床のタイプを選択する

1. プロパティパネルのタイプセレクタで床のタイプを選ぶ
   - 例：「床: 木造フロア - 200mm」「床: コンクリート - 150mm」など
2. 必要であれば「タイプ編集」から厚さ・材質を変更する

#### 2-3. 床の輪郭を描く

1. 「境界線を描く」パネルから描画ツールを選択（直線・矩形・ピックライン など）
2. 床の外周をクリックして閉じたループを作成する
   - 輪郭は**必ず閉じている**必要がある（開いていると完了できない）
3. 「**ピックウォール**」ツールを使うと、既存の壁の面に沿って輪郭を自動作成できる

**ピックウォールを使った方法（推奨）**:
1. 「境界線を描く」パネルの「**ピックウォール**」をクリック
2. 各壁をクリックして輪郭に追加していく
3. オプションバーで「壁の内面を延長」のオフセット値を設定できる（外壁内面に揃える場合は0）

#### 2-4. スケッチモードを終了する

1. スケッチ輪郭が正しく閉じていることを確認
2. 「**モード**」パネルの「**完了（緑のチェック）**」をクリック
3. ダイアログが表示された場合：「壁にアタッチしますか？」→ 壁の形状に合わせる場合は「はい」

---

### ステップ3: 屋根を作成する

屋根の作成方法は主に2種類あります。形状に応じて使い分けます。

#### 方法A: フットプリント形式（一般的な切妻・寄棟屋根）

平面図上で外周を描き、各辺に勾配を設定して屋根形状を自動生成する方法。

1. 平面図ビューで「**建築**」タブ → 「**ビルド**」パネル → 「**屋根**」→「**屋根: フットプリント**」をクリック
2. スケッチモードに切り替わる
3. 「**ピックウォール**」または「**境界線を描く**」で屋根の輪郭を描く
4. 各輪郭線を選択し、オプションバーで以下を設定:
   - **勾配を定義**: チェックを入れると、その辺が勾配面になる（切妻端部はチェックを外す）
   - **オーバーハング**: 外壁からの屋根のはね出し距離（例：500mm）
   - **勾配**: 角度または比率で設定（例：30度、または3/10）
5. 「**完了（緑のチェック）**」をクリックして屋根を生成

#### 方法B: 押し出し形式（片流れ・バレル屋根など）

立面図または断面図でプロファイル（断面形状）を描き、奥行き方向に押し出して屋根を作る方法。

1. 立面図ビューを開く
2. 「**建築**」タブ → 「**屋根**」→「**屋根: 押し出し**」をクリック
3. 参照レベルと参照オフセットを設定する
4. 屋根の断面プロファイルを描く（開いたスプラインや直線でよい）
5. 「**完了**」をクリック
6. プロパティパネルで押し出しの開始点・終了点（奥行き）を設定する

#### 壁と屋根をアタッチする

屋根を作成したら、壁の上部を屋根に合わせてカットします。

1. アタッチしたい壁を選択する（複数選択可）
2. 「修正 | 壁」タブ → 「修正ウォール」パネル → 「**アタッチ（上部/下部）**」をクリック
3. アタッチ先の屋根をクリックする
4. 壁の上端が屋根面に沿ってカットされる

---

### ステップ4: Join Geometry（要素の結合）で交差部を処理する

壁同士や壁と柱が交差する部分を、材質を考慮して自動的に結合処理する機能です。

#### 4-1. 自動結合の確認

Revitでは同じカテゴリの要素（例：壁と壁）は、交差している場合に自動的に結合されることがあります。まず、交差部を3Dビューで確認してみましょう。

#### 4-2. 手動でJoin Geometryを実行する

自動結合されていない場合や、カテゴリが異なる要素を結合する場合:

1. 「**修正**」タブ → 「**ジオメトリ**」パネル → 「**結合**」をクリック
2. 1つ目の要素（例：壁）をクリック
3. 2つ目の要素（例：柱や床）をクリック
4. 2つの要素が結合され、交差部の線がなくなる

**結合の解除**:
- 「**結合**」ボタンのドロップダウンから「**結合解除**」を選択し、結合している要素を順にクリック

#### 4-3. 結合順序の変更

同じ位置に複数の要素が重なっている場合、どの材質が前面に来るかを「結合順序」で制御できます。

1. 「**修正**」タブ → 「**ジオメトリ**」パネル → 「**結合**」のドロップダウン → 「**結合の順序を切り替え**」
2. 結合している要素をクリックすると、どちらの材質が前面に来るかが反転する

---

## Tips

- 壁を描く際は「位置線」の設定が重要。「壁の中心線」だと壁の中心を基準に描かれるため、外形寸法で描く場合は「コア面: 外部」などに変更する
- スケッチモード中は輪郭が**マゼンタ（紫）色**で表示される。赤くなっている場合は輪郭に問題がある
- 床の「ピックウォール」使用時、同じ外壁を複数回クリックすると重複ラインになるため注意する
- 屋根のオーバーハング値はフットプリント輪郭の各辺を選択した状態でのみ変更できる
- 壁に窓やドアを配置した後でもJoin Geometryは正常に機能する

---

## 注意事項

- スケッチモード（床・屋根など）では、輪郭が閉じていないと「完了」できない。輪郭の端点が正確に接続されているか確認する
- 「壁にアタッチ」を実行すると、後から屋根を移動した場合に壁も連動して変形する
- Join Geometryは**同じフェーズ**の要素間でのみ機能する
- 屋根の押し出し形式では、プロファイルに不正な形状（自己交差など）があるとエラーになる
- ※Revit 2022以降は、床の境界編集時のUIが一部変更されており、「境界線を描く」パネルのアイコン配置が変わっている場合がある
