# 平面図・立面図・断面図の作り方

Revitでは、3Dモデルから平面図・立面図・断面図を自動生成できます。これはRevitの最大の強みのひとつです。この記事では、各種ビューの作成方法と表示設定の基本を解説します。

---

## この記事でわかること

- 平面図の作成方法とレベルとの関係
- 立面図マーカーの配置と方向の調整方法
- 断面図マーカーの配置と表示範囲の設定方法
- 3Dビューの作成とセクションボックスの使い方
- ビューテンプレートを使ったビュー設定の一括適用方法

---

## 手順

### ステップ1: 平面図を作成する

#### 平面図とレベルの関係

Revitの平面図は「レベル」と1対1で対応しています。レベルとは建物の各階の高さを定義する水平面で、レベルを追加すると自動的に対応する平面図が生成されます（設定による）。

平面図は、そのレベルの高さから「ビュー範囲」で設定した高さで水平にモデルを切断して見下ろした図です。

#### 1-1. 既存のレベルから平面図を作成する

レベルは作成済みだが平面図がない場合:

1. 「**表示**」タブ → 「**作成**」パネル → 「**平面図**」→「**平面図**」をクリック
2. 「新しい平面図」ダイアログが表示される
3. 平面図を作成したいレベル（例：レベル3）を選択する（複数選択可）
4. 「**OK**」をクリック
5. プロジェクトブラウザの「ビュー」→「平面図」に新しいビューが追加される

#### 1-2. レベルを追加して平面図を作成する

1. 立面図ビューを開く
2. 「**建築**」タブ → 「**基準面**」パネル → 「**レベル**」をクリック
3. 既存レベルの横にカーソルを合わせ、水平線をクリックして引く
4. レベル高さは数値を直接入力するか、後からプロパティパネルで変更する
5. 「平面図を作成しますか？」ダイアログが表示されたら「**はい**」をクリック

#### 1-3. ビュー範囲を調整する

平面図で表示される高さの範囲は「ビュー範囲」で設定します。

1. 平面図ビューを開いた状態で、何も選択しない
2. プロパティパネルの「範囲」セクションにある「**ビュー範囲**」の「編集」ボタンをクリック
3. 以下の値を設定する:

| 設定項目 | 説明 | 一般的な設定例 |
|---------|------|-------------|
| 上限 | 切断面より上の表示上限 | 現在レベル + 4000mm |
| 切断面 | 水平に切断する高さ | 現在レベル + 1200mm |
| 下限 | 表示する最低高さ | 現在レベル |
| ビュー深度 | 下限より下にある要素を表示する高さ | 現在レベル - 300mm |

4. 「**OK**」をクリック

---

### ステップ2: 立面図マーカーを配置・調整する

#### 2-1. 立面図の仕組み

Revitには最初から東・西・南・北の4方向の立面図が存在します。これらは平面図上の「立面図マーカー」（正方形の中に矢印があるシンボル）によって制御されています。

#### 2-2. 新しい立面図を作成する

任意の方向や位置の立面図を追加したい場合:

1. 平面図ビューを開く
2. 「**表示**」タブ → 「**作成**」パネル → 「**立面図**」をクリック
3. ビュー上でクリックして立面図マーカーを配置する
   - マーカーを配置すると、プロジェクトブラウザに新しい立面図が自動追加される

#### 2-3. 立面図の方向と範囲を調整する

1. 平面図ビューで立面図マーカーをクリックして選択する
2. マーカーの周囲に青い矩形（クロップ領域）が表示される
3. 矢印の方向を変えるには: マーカーを選択した状態でハンドルをドラッグして回転するか、プロパティパネルで「**回転**」を変更する
4. 表示範囲を変えるには: クロップ領域の端にある青い矢印をドラッグして広げる

**立面図の切断面を確認**:
- 立面図マーカーの正方形の各辺には小さなチェックボックスがある
- チェックを入れた辺に対して立面図が生成される（最大4方向）

---

### ステップ3: 断面図マーカーを配置・範囲設定する

#### 3-1. 断面図を作成する

1. 平面図・立面図・別の断面図のいずれかのビューを開く
2. 「**表示**」タブ → 「**作成**」パネル → 「**断面図**」をクリック
3. 断面図の切断方向に沿って、起点から終点へドラッグ（または2点クリック）する
4. 断面図マーカーが配置され、プロジェクトブラウザに新しい断面図ビューが追加される

#### 3-2. 断面図の表示範囲を設定する

断面図マーカーを選択すると、表示範囲を制御するハンドルが表示されます。

| ハンドル | 位置 | 機能 |
|---------|------|------|
| 両端の矢印ハンドル | マーカー両端 | 断面切断線の長さ（横方向の範囲）を調整 |
| 遠端のライン | 奥行き方向の遠端 | 断面図の奥行き（深さ）を調整 |
| 近端のライン | 奥行き方向の近端 | 断面図の手前の切断位置を調整 |

**ジョグ（折れ曲がり断面）**:
断面図マーカーを選択し、中央のコントロールをクリックするとマーカーにジョグ（折れ曲がり）を追加できます。これにより、1つの断面図で複数の部分を表示できます（※Revit 2019以降）。

#### 3-3. 断面図ビューを開く

- プロジェクトブラウザの「断面図」フォルダをダブルクリック
- または、平面図上の断面図マーカーをダブルクリックして直接開く

---

### ステップ4: 3Dビューを作成してセクションボックスを使う

#### 4-1. デフォルト3Dビューを開く

- クイックアクセスツールバーの「**3Dビュー**」ボタン（家のアイコン）をクリック
- または「表示」タブ → 「作成」パネル → 「3Dビュー」→「デフォルト3D」

#### 4-2. 新しい3Dビューを作成する

複数の3Dビューを使い分けたい場合:

1. 「**表示**」タブ → 「**作成**」パネル → 「**3Dビュー**」→「**カメラ**」または「**3D（アイソメ）**」
2. 「カメラ」はパース（透視投影）ビューを作成する
3. 「3D（アイソメ）」はデフォルトのアイソメトリックビューと同様の視点で新しいビューを作成する

#### 4-3. セクションボックスでモデルを断面表示する

セクションボックス（切断ボックス）は、3Dビューで指定した直方体の範囲内だけを表示する機能です。

**セクションボックスを有効にする**:
1. 3Dビューを開き、何も選択しない
2. プロパティパネルの「範囲」セクションの「**切断ボックス**」にチェックを入れる
3. モデルの周囲に半透明の青い直方体が表示される

**セクションボックスの範囲を調整する**:
1. 切断ボックス（青い直方体）を選択する
2. 各面に青い矢印ハンドルが表示される
3. ハンドルをドラッグして切断面を移動させる
4. 切断面を内側に移動させるとモデルが断面表示になる

**特定の部分だけを表示する**:
セクションボックスを平面図ビューのクロップ領域と連動させることもできます。

---

### ステップ5: ビューテンプレートを適用する

ビューテンプレートは、尺度・詳細レベル・表示スタイル・フィルターなどの設定をまとめて保存し、複数のビューに一括適用する機能です。

#### 5-1. 既存のビューテンプレートを適用する

1. テンプレートを適用したいビューを開く（またはプロジェクトブラウザで選択）
2. プロパティパネルの「**ビューテンプレート**」の欄をクリック
3. テンプレートの一覧が表示されるので、適用したいテンプレートを選択する
4. 「**OK**」をクリック

#### 5-2. 現在のビューからテンプレートを作成する

1. ソースとなるビューを開き、表示設定を整える
2. 「**表示**」タブ → 「**グラフィックス**」パネル → 「**ビューテンプレート**」→「**現在のビューからテンプレートを作成**」をクリック
3. テンプレート名を入力して「**OK**」
4. 作成したテンプレートを他のビューに適用できるようになる

#### 5-3. ビューテンプレートを編集する

1. 「**表示**」タブ → 「**グラフィックス**」パネル → 「**ビューテンプレート**」→「**ビューテンプレートの管理**」
2. テンプレート一覧から編集したいテンプレートを選択
3. 右側のプロパティで各設定を変更する

**ビューテンプレートで制御できる主な設定**:

| 設定項目 | 内容 |
|---------|------|
| 表示/グラフィックス | 線の色・パターン・透明度など |
| 詳細レベル | 粗・中・細の切り替え |
| パーツの表示 | パーツ機能の表示/非表示 |
| ビジュアルスタイル | ワイヤーフレーム・隠線・シェーディングなど |
| 尺度 | ビューの尺度（1:100など） |
| フィルタ | カテゴリフィルタの設定 |

## Tips

- 断面図マーカーは平面図上では見えていても、「表示」→「グラフィックス」設定でマーカーを非表示にできる
- 3Dビューでセクションボックスを使いながら平面図のビュー範囲を調整すると、モデルの空間把握がしやすい
- ビューテンプレートを適用したビューでは、テンプレートで管理されているプロパティはプロパティパネルで直接変更できなくなる。変更したい場合はテンプレートの割り当てを解除するか、テンプレート自体を編集する
- ビューを大量に作成するとプロジェクトが重くなるため、使わなくなったビューは削除するか「作業ビュー」フォルダに整理する習慣をつける
- ※Revit 2021以降では、複数のビューを選択してビューテンプレートを一括適用できる

---

## 注意事項

- レベルを削除すると、そのレベルに紐づく平面図も削除される場合があるため注意する
- 立面図マーカーを削除すると、対応する立面図ビュー自体も削除される
- ビューテンプレートで「尺度」を管理している場合、個別ビューの尺度は変更できなくなる
- セクションボックスはあくまでビュー内の表示設定であり、モデル自体を変更するものではない
- 断面図の「近端（Far Clip Offset）」の距離を短くしすぎると、表示されるべき要素が消えることがある
