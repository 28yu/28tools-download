# Revitの画面構成を理解する

Revitを初めて起動すると、CADソフトとは異なる独特のインターフェースに戸惑うことがあります。この記事では、Revitの主要な画面要素をひとつずつ丁寧に解説します。画面構成をしっかり把握することで、その後の操作がスムーズになります。

---

## この記事でわかること

- プロジェクトブラウザの役割とビューの切り替え方
- プロパティパネルで要素の情報を確認・編集する方法
- リボン（タブ・パネル）の構成と主要機能の場所
- ビューウィンドウでのズーム・パン・回転操作
- ステータスバーと情報バーの読み方

---

## 手順

### ステップ1: 画面全体のレイアウトを把握する

Revitを起動してプロジェクトを開くと、次の要素が画面に表示されます。

| 画面要素 | 位置 | 役割 |
|----------|------|------|
| アプリケーションメニュー | 左上の「R」マーク | 新規作成・保存・印刷などのファイル操作 |
| クイックアクセスツールバー | 最上部 | 保存・元に戻すなど頻繁に使う操作 |
| リボン | 上部 | タブごとに分類されたコマンド群 |
| プロジェクトブラウザ | 左側パネル | 全ビュー・シート・ファミリの一覧 |
| プロパティパネル | 左側パネル（ブラウザの上） | 選択要素のプロパティ表示・編集 |
| ビューウィンドウ | 中央の大きな領域 | モデルを表示・編集する作業エリア |
| ビューコントロールバー | ビュー下部 | 尺度・詳細レベル・表示モードの設定 |
| ステータスバー | 最下部 | 現在の操作の説明・ヒント表示 |

---

### ステップ2: プロジェクトブラウザを使いこなす

プロジェクトブラウザは、プロジェクト内のすべてのビューや要素をツリー形式で管理するパネルです。

**主な内容**:
- **ビュー**: 平面図・立面図・断面図・3Dビュー・天井伏図など
- **凡例**: 凡例ビュー
- **集計表/数量**: 数量表・マテリアル集計表
- **シート**: 印刷用の図面シート
- **ファミリ**: プロジェクト内で使用しているファミリ
- **グループ**: モデルグループ・詳細グループ

**ビューを開く方法**:
1. プロジェクトブラウザ内の目的のビュー名をダブルクリックする
2. ビューウィンドウに選択したビューが表示される

**ビューを検索する方法**:
- ブラウザ上部の検索ボックス（虫眼鏡アイコン）にキーワードを入力すると、該当ビューを絞り込める（※Revit 2020以降）

**プロジェクトブラウザが消えた場合**:
- 「表示」タブ → 「ウィンドウ」パネル → 「ユーザーインターフェース」→「プロジェクトブラウザ」にチェックを入れる

---

### ステップ3: プロパティパネルを活用する

プロパティパネルは、現在選択している要素のプロパティ（属性情報）を表示・編集するためのパネルです。

**何も選択していない状態**: アクティブなビュー自体のプロパティが表示される
**要素を選択した状態**: 選択した要素（壁・柱・ドアなど）のプロパティが表示される

**プロパティパネルの構成**:

```
[タイプセレクタ]     ← 要素のタイプ名が表示される（クリックで変更可）
[タイプ編集ボタン]   ← タイプのパラメータを編集するダイアログを開く
─────────────────
インスタンスプロパティ（個々の要素ごとの設定）
  ・制約（レベル、オフセットなど）
  ・寸法（長さ、面積など、読み取り専用）
  ・識別データ（コメント、マークなど）
─────────────────
[適用ボタン]        ← 変更をモデルに反映する
```

**タイププロパティとインスタンスプロパティの違い**:
- **タイププロパティ**: 同じタイプの全要素に影響する（壁の厚さなど）
- **インスタンスプロパティ**: 選択した要素のみに影響する（壁の高さ・位置など）

**プロパティパネルが消えた場合**:
- 「表示」タブ → 「ユーザーインターフェース」→「プロパティ」にチェックを入れる

---

### ステップ4: リボン（タブ・パネル）の構成を理解する

リボンは画面上部に配置されており、タブを切り替えることで関連するコマンドにアクセスします。

**主なタブと用途**:

| タブ名 | 主な用途 |
|--------|---------|
| 建築 | 壁・床・屋根・ドア・窓などの建築要素の作成 |
| 構造 | 柱・梁・スラブなどの構造要素の作成 |
| システム | 設備（MEP）関連の要素の作成 |
| 挿入 | リンクファイル（他のRevit・DWGなど）の挿入・読み込み |
| 注釈 | 寸法・テキスト・タグ・記号などの注釈要素の追加 |
| 解析 | 建物の性能分析（エネルギー・構造など） |
| マスと敷地 | マスモデルや敷地の作成 |
| コラボレーション | ワークシェアリング（複数人での共同作業） |
| 表示 | ビューの作成・グラフィックス設定・ウィンドウ管理 |
| 管理 | プロジェクト設定・パラメータ・フェーズ管理 |
| アドイン | サードパーティのアドインツール |
| 修正 | 要素を選択したときに表示される編集コマンド |

**コンテキストタブ（修正タブ）**:
要素を選択したり、コマンドを実行中のときは「修正」タブが自動的に表示され、その操作に適したコマンドが表示されます。

---

### ステップ5: ビューウィンドウの操作を習得する

ビューウィンドウ内でのナビゲーション操作は頻繁に使うため、しっかり覚えておきましょう。

**ズーム操作**:
- **マウスホイールを前に回す**: ズームイン（拡大）
- **マウスホイールを後ろに回す**: ズームアウト（縮小）
- **ZE キー（またはZF）**: 全体表示（モデル全体が画面に収まるように調整）
- **ZZ キー**: ズームイン（選択範囲を拡大）

**パン（画面移動）操作**:
- **マウスホイールを押しながらドラッグ**: 画面をスクロール（パン）

**回転操作（3Dビューのみ）**:
- **Shift + マウスホイールを押しながらドラッグ**: 視点を自由に回転
- **ViewCubeをドラッグ**: 画面右上のViewCubeをドラッグして回転
- **ViewCubeの面・辺・角をクリック**: その方向の正面ビューに切り替え

**ビューの分割表示**:
- 「表示」タブ → 「ウィンドウ」パネル → 「タイル表示」で複数ビューを同時に表示できる

---

### ステップ6: ステータスバーと情報バーを活用する

**ステータスバー（画面最下部）**:
現在実行中のコマンドの説明や次に行うべき操作のヒントが表示されます。コマンド実行中は必ずここを確認する習慣をつけましょう。

例: 壁コマンド実行中 →「壁の起点をクリックしてください」と表示

**ビューコントロールバー（ビュー下部）**:
各ビューの表示設定を素早く変更できるバーです。

| アイコン | 機能 |
|---------|------|
| 1:100 などの数字 | 尺度の設定 |
| 粗・中・細 | 詳細レベルの切り替え |
| ワイヤーフレームなどのアイコン | ビジュアルスタイルの切り替え |
| 太陽アイコン | 日照設定 |
| 影アイコン | 影表示のオン/オフ |
| メガネアイコン | クロップ領域の表示/非表示 |

---

## Tips

- プロジェクトブラウザとプロパティパネルが消えた場合は、「表示」タブ →「ユーザーインターフェース」から復元できる
- タブ内のコマンドが灰色（非アクティブ）の場合は、先にビューを開くか要素を選択する必要がある
- リボン右端の小さな矢印（ダイアログランチャー）をクリックすると、より詳細な設定ダイアログが開く
- ViewCubeの「ホーム」アイコンをクリックすると、初期の3D視点に戻れる
- ショートカットキー「KS」でキーボードショートカット一覧を表示・カスタマイズできる

---

## 注意事項

- プロパティパネルの変更は「適用」ボタンを押すまでモデルに反映されない場合がある
- タイププロパティを変更すると、同じタイプを使っている**すべての要素**に影響するため注意が必要
- リボンのタブ構成はRevitのバージョンや使用するライセンス（建築・構造・MEP）によって一部異なる
- ビューウィンドウのタブが増えすぎると動作が重くなることがあるため、使わないビューは閉じることを推奨する
