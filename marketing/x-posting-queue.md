# X 投稿キュー（Buffer / Typefully 流し込み用）

作成: 2026-07 ／ 元データ: `marketing/x-promotion-plan.md`（UTM・3タグ反映済み）

> **使い方**: 下の順番どおりに、各ブロックを Typefully / Buffer の下書きに1本ずつ貼り付け、
> 指定の曜日・時刻でスケジュール（キュー投入）してください。投稿はツールが自動で行い、
> あなたは**通知が来たら承認/スキップを選ぶだけ**。GIF添付が推奨のものは注記あり。

## ツール別セットアップ（どちらか）

### ◎ Typefully（おすすめ・X下書きに最適）
1. typefully.com に登録 → Xアカウント（@28tools_com）を連携
2. 右上「設定 → Scheduling」で**投稿枠（時間帯）**を登録：平日 08:00 / 12:15 / 20:30・21:00、土日 11:00・21:00 など
3. 下の各ブロックを「New draft」に貼り付け → **Schedule**（時刻指定）or **Add to Queue**（次の空き枠へ自動割当）
4. スマホアプリを入れると、公開前に通知で確認できる（承認/編集/スキップ）
5. **固定A（No.A）は最初に投稿 → X側でプロフィールにピン留め**

### ○ Buffer（シンプルなキュー運用）
1. buffer.com に登録 → Xを連携
2. 「Posting Schedule」で上と同じ時間帯を登録
3. 各ブロックを「Add to Queue」。Bufferが枠順に自動投稿
4. モバイルアプリで公開前にキューを確認・並べ替え・削除できる

> ⚠️ どちらも**1日1〜2本**を守る（連投しない）。GIF/画像は各ツールの投稿画面で添付。

# 返信サポート（リプ/いいねへの対応）

**運用**: 自動化はしない（凍結・低品質化リスク）。来たリプを私（Claude Code）か Claude チャットに貼れば、
最適な返信文を即提案します。まずは下のテンプレをベースに、相手に合わせて一言だけ変えるのが速い。

### よく来るパターン別・返信テンプレ
| 相手の反応 | 返信テンプレ（一言変えて使う） |
|---|---|
| 「便利そう」「使ってみます」 | ありがとうございます！ぜひ触ってみてください🙏 もし詰まったら気軽にリプください。 |
| 使い方の質問 | 〔手順を一言で〕でいけます！ 他に不明点あればいつでもどうぞ。 |
| 感想・あるある共感 | それ、ほんとあるあるですよね😅 実は〔別の使いどころ〕でも効きますよ。 |
| 要望・こうしてほしい | フィードバックありがとうございます！ 〔検討する/直しました〕。助かります🙏 |
| 不具合報告 | 教えていただき感謝です！ 〔状況確認 or 修正予定〕。DMで詳細いただけると助かります。 |
| 「無料なの？」 | はい、無料です👍 アドインはDL、ブラウザツールはインストール不要でそのまま使えます。 |
| ほめ/応援 | 励みになります…！ これからも作図が少しでもラクになるもの作っていきます🛠️ |

### いいねをくれた人への動き
- 返信はできないので、**プロフィールを見て建築/BIM実務者ならフォロー**＋その人の投稿に1いいね/リプ。

### 返信の原則
- 来たリプは**全部いいね＋一言返信**（特に立ち上げ期）。投稿後30〜60分は張り付くと伸びる。
- 同じ定型文の連投はしない（毎回少し変える）。URLは相手の悩みに直接効く時だけ。
