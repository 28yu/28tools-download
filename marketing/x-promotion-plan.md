## 4. エンゲージ戦略

### 4-1. 絡む相手像（フォロー・リプライ対象）
- **建築BIM系の実務者**: Revit / ArchiCAD ユーザー、設計事務所・ゼネコン・組織設計の中の人。
- **BIMマネージャー / BIMコーディネーター**を名乗る人。
- **構造設計者・施工図オペレーター**（PDF→Excel・型枠・梁色分けの刺さり先）。
- **建築系の情報発信者**（Revitの使い方・Dynamo・アドイン開発を発信する個人）。
- **建築学生・BIM学習者**（無料ツールは学習層に強い＝拡散されやすい）。
- **やり方**: 上記が使うハッシュタグ（下記一覧）や「Revit 使い方」などで検索 → 有益な投稿に**中身のあるリプ**を返す。無言フォロー爆撃はしない。

### 4-2. 有益リプの型（コピペ改変用）

**型① 共感＋一次情報の補足**（宣伝しない）
```
分かります、その作業ほんと時間かかりますよね。自分は〔具体的な回避策〕でやってます。〔一言の理由〕なので手戻りが減りました。
```

**型② 質問に答える＋そっと導線**（相手が困りごとを投稿している時のみ）
```
それ、〔手順1〕→〔手順2〕でいけます。もし手作業がきついようなら、同じことを自動化する無料ツールも作ってあるので使ってみてください→ URL
```
※**相手の悩みに直接効く時だけ**URLを出す。無関係な投稿への貼り付けはスパム扱い。

**型③ ノウハウ提供（引用リポスト）**
```
これ大事。補足すると〔規格/仕様の根拠〕なので〔結論〕です。〔具体例〕
```

**型④ 感謝・称賛（関係構築）**
```
この知見ありがたいです。〔具体的にどこが役立ったか〕。保存しました🙏
```

### 4-3. エンゲージの原則（凍結・嫌われ回避）
- **1日のリプは10〜20件まで**、フォローは**1日20〜30人まで**を目安に（急激な数は自動制限対象）。
- **同じ定型文の連投をしない**（毎回少し変える）。
- **相手の投稿を読まずにURLを貼らない**。価値提供 → 関係 → たまに導線、の順。
- 自分の投稿には**リプが来たら必ず返す**（会話が続くとインプレッションが伸びる）。
- 良い他人の投稿は**引用リポストで自分の見解を足す**（ただ拡散よりフォローされやすい）。
