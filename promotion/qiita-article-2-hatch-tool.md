## このツールでできること

### 対応パターン（6種類）

| パターン | 用途例 |
|---------|-------|
| **斜線** | 構造図の断面表現、敷地境界線 |

## 使い方（3ステップ）

### ステップ1: パターンを選ぶ

![パターン選択画面](https://28tools.com/images/portal/icon-hatch.png)

ツールにアクセスし、6種類のパターンから選択します。

### ステップ2: パラメータを設定

各パターンごとに設定できるパラメータが表示されます。

#### 斜線・網掛け
- **角度**: 0〜180度
- **間隔**: 1〜100mm
- **破線設定**: 実線 / 破線（線長・空白長）

#### ドット
- **間隔**: 1〜100mm
- **ドットサイズ**: 0.5〜10mm

#### 芋目地・馬目地
- **X方向サイズ**: 1〜500mm
- **Y方向サイズ**: 1〜500mm
- **目地あり/なし**: チェックボックスで切り替え
- **目地サイズ**: X方向・Y方向それぞれ設定可能

#### RC
- **線内間隔**: 線内部の間隔
- **グループ間隔**: グループ間の間隔

### ステップ3: プレビュー → ダウンロード

1. 設定を変更すると**リアルタイムプレビュー**が表示
2. 「Revit用ダウンロード（モデル）」または「AutoCAD用」をクリック
3. .patファイルがダウンロードされる

## 実際の使用例

### 例1: 芋目地（100x200mm、目地5mm）

**設定**:
- X方向: 100mm
- Y方向: 200mm
- 目地あり: ON
- 目地X: 5mm
- 目地Y: 5mm

**出力ファイル名**: `芋目地_100x200x5x5.pat`

**Revitでの使用方法**:
1. .patファイルを以下のフォルダに配置：
   ```
   C:\ProgramData\Autodesk\RVT 2024\Libraries\日本\
   ```
2. Revitを起動
3. 塗潰しパターン → 「芋目地_100x200x5x5」を選択

---

### 例2: 斜線（45度、間隔10mm、破線）

- 間隔: 10mm
- 破線: ON
- 線長: 5mm
- 空白長: 2mm

**用途**: 構造図の断面ハッチング

---

### 例3: RCコンクリート

**設定**:

## 技術的なポイント

### Shift-JISエンコーディング

Revit/AutoCADはWindows環境でShift-JISエンコーディングを期待するため、パターン名に日本語を使用する場合は**Shift-JISでエンコード**しています。

このツールでは、ブラウザ上でShift-JISエンコーディングを実装しており、ダウンロードした.patファイルをそのままRevit/AutoCADで使用できます。

### .pat形式の構造

.patファイルは以下の形式で記述されます：

```
*芋目地_100x200x5x5,芋目地（100x200mm、目地5x5mm）
0,      0,       0,       100,     200
90,     0,       0,       200,     100
0,      52.5,    0,       100,     200
90,     0,       102.5,   200,     100
```

**各行の意味**:
- `*パターン名,説明`
- `角度, X原点, Y原点, delta-x, delta-y`

このツールでは、ユーザーが設定したパラメータから自動的にこの形式を生成します。

## よくある質問

### Q1: ダウンロードしたファイルはどこに保存すればいい？

**Revitの場合**:
```
C:\ProgramData\Autodesk\RVT [バージョン]\Libraries\日本\
```

**AutoCADの場合**:
```
C:\Users\[ユーザー名]\AppData\Roaming\Autodesk\AutoCAD [バージョン]\ja-JP\Support\
```

- .patファイルが正しいフォルダに配置されているか
