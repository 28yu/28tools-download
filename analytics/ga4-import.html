📊 GA4 歴史データ インポート

Google Analytics 4 の過去データを Supabase に取り込み、ダッシュボードに歴史データを表示します。

Step 1 — Supabase 接続

Step 2 — Google Apps Script でデータをエクスポート

1
GA4 プロパティ ID を確認
Google Analytics を開く → 左下の 管理（歯車） → プロパティの設定 → 「プロパティ ID」の数字（例: 123456789）をメモ
2
Google スプレッドシートで Apps Script を実行
新しいスプレッドシートを開く → メニュー「拡張機能」→「Apps Script」→ 以下をコピーして貼り付け → PROPERTY_ID を自分のIDに変更 → 実行 
// GA4 Data API を使って歴史データを CSV エクスポート
// ※ 初回実行時に「Google Analytics Data API」サービス追加が必要:
//    左メニュー「サービス」→「Google Analytics Data API」→追加

function exportGA4() {
  const PROPERTY_ID = '123456789'; // ← あなたのGA4プロパティIDに変更

  const report = AnalyticsData.Properties.runReport({
    dateRanges: [{ startDate: '2020-01-01', endDate: 'today' }],
    dimensions: [
      { name: 'date' },
      { name: 'pagePath' },
      { name: 'country' },
      { name: 'browser' },
      { name: 'operatingSystem' },
      { name: 'language' },
    ],
    metrics: [
      { name: 'screenPageViews' },
      { name: 'sessions' },
    ],
    limit: 250000,
  }, 'properties/' + PROPERTY_ID);

  const rows = [['date','page','country','browser','os','lang','views','sessions']];
  (report.rows || []).forEach(row => {
    rows.push([
      ...row.dimensionValues.map(d => d.value),
      ...row.metricValues.map(m => m.value),
    ]);
  });

  const csv = rows.map(r =>
    r.map(c => '"' + String(c).replace(/"/g, '""') + '"').join(',')
  ).join('\n');

  const blob = Utilities.newBlob(csv, 'text/csv', 'ga4_export.csv');
  const file = DriveApp.createFile(blob);
  Logger.log('✅ CSV 作成完了: ' + file.getUrl());
  // → Google ドライブに「ga4_export.csv」が作成される
}
3
CSV をダウンロード
実行ログに表示された Google ドライブの URL を開き、 ダウンロード して ga4_export.csv を保存

Step 3 — Supabase にテーブルを作成

Supabase の SQL Editor で以下を実行してください（初回のみ） 
CREATE TABLE IF NOT EXISTS ga4_history (
  id bigserial PRIMARY KEY,
  date date NOT NULL,
  page text,
  country text,
  browser text,
  os text,
  lang text,
  views integer DEFAULT 0,
  sessions integer DEFAULT 0
);
CREATE INDEX IF NOT EXISTS ga4_history_date_idx ON ga4_history (date);
CREATE INDEX IF NOT EXISTS ga4_history_page_idx ON ga4_history (page);
ALTER TABLE ga4_history ENABLE ROW LEVEL SECURITY;
CREATE POLICY IF NOT EXISTS "allow anon read" ON ga4_history FOR SELECT TO anon USING (true);
GRANT SELECT ON public.ga4_history TO anon;
GRANT SELECT ON public.ga4_history TO authenticated;
GRANT ALL ON public.ga4_history TO service_role;
GRANT ALL ON SEQUENCE public.ga4_history_id_seq TO service_role;
